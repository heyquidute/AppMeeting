1. Criei o design do app pelo figma

2. Criei o app pelo prompt: npx create-expo-app meeting --template
-> blank (typescript)

3. Adicionei "esModuleInterop": true no arquivo tsconfig.json

4. Instalei o npm install styled-components depois de reinstalar a react-dom e colocar manualmente a versão 18.2.0 nas dependências
-> foi luta, mas deu certo

5. Instalar as tipagens do ts para usar o ts junto com o styled-components
-> npm install --save-dev @types/styled-components @types/styled-components-react-native

6. Para usar o styled-components precisa importar [import styled from 'styled-components/native';]

7. Exemplo de estilização com o styled-components: 
[const Container = styled.View`
  flex: 1;
  background-color: aliceblue;
  justify-content: center;
  align-items: center;
`;]

8. Criando um estilo global:
  - Pasta theme > arq index.ts > 'export default{...}' 
  - Tipagem para parar de dar erro no theme:crianção da pasta @types > arq styled.d.ts
  - Instalação do google fonts: npx expo install expo-font @expo-google-fonts/roboto


9. Depois de configurar os estilos, comecei fazendo a tela principal, depois a tela de cadastro e o modal do 'selecione uma categoria'.
-> aprendendo a usar o useState para gerenciamento de estado e de comunicação de "pai pra filha"

10. Instalei a biblioteca do reack hook forms para melhor performance do formulário
-> npm install react-hook-form

11. Instalei o React Navigation para navegação entre telas
-> npm install @react-navigation/native
-> npm install @react-navigation/native-stack
-> npx expo install react-native-screens react-native-safe-area-context
-> npm install @react-navigation/bottom-tabs

12. Criei uma fake api basico com json serve e vou conectar ela com o app pelo axio
-> json-server --watch db.json --port 3000 (para ativar a "api")
-> npm install axios (instalação do axios)

13. Instalar a lib data-fns, para formatação de dados para datas
-> npm install date-fns --save
